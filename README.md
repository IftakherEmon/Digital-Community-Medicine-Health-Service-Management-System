# Digital Community Medicine & Health Service Management System

## Project Overview
This is a C# Windows Forms based healthcare management system developed as an academic project.
The system connects patients, doctors, pharmacists, diagnostic labs, and administrators
under one digital platform.

## Key Features
- Patient Registration & Login
- Doctor Appointment Booking
- Online Medicine Ordering
- Prescription Verification by Pharmacist
- Emergency Request & Ambulance Service
- Complaint Submission & Management
- Diagnostic Lab Test Booking
- Follow-up Reminder & Notification System
- Admin Dashboard for System Control

## Technologies Used
- Language: C#
- Framework: .NET (Windows Forms)
- Database: Microsoft SQL Server
- IDE: Visual Studio 2022

## Database Design
- Fully normalized (3NF)
- 30+ relational tables
- Foreign key constraints applied

## User Roles
- Admin
- Patient
- Doctor
- Pharmacist

## How to Run the Project
1. Open the `.sln` file in Visual Studio 2022
2. Restore database using provided SQL scripts
3. Update SQL connection string if needed
4. Run the project
